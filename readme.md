# 🍅 Pomodoro Timer

一个精致、无边框的桌面番茄钟，基于 Python 和 Tkinter 构建。  
支持自定义工作时长、自动切换休息、历史统计，以及细腻的动画效果。

![screenshot](screenshot.png) <!-- 可替换为实际截图 -->

## ✨ 功能特性

- 经典的番茄工作法循环（工作 → 短休息 → 长休息）
- 自定义工作时长、短休息和长休息时间
- 无边框窗口，圆角卡片设计，支持拖拽移动
- 计时阶段切换时平滑的颜色渐变与文字脉冲动画
- 每日番茄完成数量本地记录（JSON）
- 历史记录查看窗口
- 系统设置窗口（滑块调整时长）
- 代码分层清晰，核心逻辑与 UI 完全解耦，便于二次开发

## 📁 项目结构

```
pomodoro/

│   ├── app_window.py        # 无边框主窗口、拖拽、标题栏

│   └── history_window.py    # 历史记录窗口

## 🚀 快速开始

### 环境要求
- Python 3.7+
- 仅使用标准库 `tkinter`、`json`、`os`、`datetime`、`ctypes`，无需额外安装。

### 运行方法
在终端中进入 `pomodoro/` 所在的父目录，执行：

```bash
python pomodoro/main.py
```

或者直接在 `pomodoro/` 目录内运行：

```bash
python main.py
```

程序启动后，你会看到一个无边框的圆角窗口，点击 **▶ 开始工作** 即可开始计时。

## 🎮 使用说明

- **开始/暂停**：点击主按钮或按对应快捷键（暂无全局热键）
- **重置**：回到工作时长，清空当前进度
- **调整时间**：通过窗口上方的 `−` `+` 按钮修改工作/休息时长（计时中需先暂停）
- **移动窗口**：按住顶部标题栏 `🍅 番茄钟` 拖动
- **关闭程序**：点击右上角 `✕`
- **菜单**：点击标题栏左侧的 `☰`，可打开 **系统设置** 或 **历史记录**
- **系统设置**：用滑块精细调节时长，点击“应用”立即生效
- **历史记录**：查看每天的番茄完成个数

## 🎨 自定义

所有颜色、字体大小、窗口尺寸均集中在 `ui/styles.py`，修改后重启即可生效。  
动画参数可在 `ui/animations.py` 中调整（脉冲幅度、渐变速度等）。

## 📝 数据存储

完成的番茄会记录到 `pomodoro/tomato_data.json`，格式如下：

```json
{
    "2026-06-27": 3,
    "2026-06-28": 5
}
```
